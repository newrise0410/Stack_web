# 29cm.co.kr 홈 — 구조 토폴로지 (구조 이식용)

정찰일: 2026-07-13 · 데스크톱 1440 / 모바일 390 · Playwright
목적: **레이아웃/컴포넌트 패턴만** 추출해 Stack N' Stak 홈에 이식.
29cm의 브랜딩·캠페인 이미지·카피·상품은 이식하지 않음(전부 우리 자산으로 대체).

## 섹션 리듬 (위 → 아래)

1. **헤더** — 로고(좌) · 유틸 아이콘(MY PAGE/LIKE/BAG/LOGIN, 우) · 대형 타이포 주 메뉴행
   (Special-Order / Showcase / PT / Magazine) · 카테고리행(BEST/…) · 검색.
   스크롤 시 대형 타이포행이 접히며 헤더가 축소(compact)됨.
2. **히어로 = 3열 캠페인 타일** — 뷰포트 폭을 3분할, 각 타일 ≈ 603×723(4:5 세로),
   xl에서 좌우 화살표 네비. 각 타일 = 풀블리드 이미지 + 하단 캠페인 문구.
3. **3열 에디토리얼 모듈** — 각 열 = [상단 이미지 카드] + [제목] + [서브카피] +
   [상품 리스트]. 상품 행 = 썸네일 · 브랜드 · 상품명 · 할인% · 가격 · 하트/찜수.
4. (하위: 랭킹 레일 · 큐레이션 가로 레일 · 배너 — 가상화 무한 피드, 이식 범위 밖)

## 이식 대상 (우리 홈에 없는 29cm 시그니처)

- **[A] 3열 캠페인 히어로** ← 현재는 단일 풀블리드 히어로. → `home/HeroCampaign.jsx`
- **[B] 3열 에디토리얼 모듈**(이미지 카드 + 상품 리스트) ← 없음. → `home/EditorialModule.jsx`
- 카테고리 퀵내비(선택) — 히어로와 모듈 사이.

## 유지 (기존 홈의 양호한 섹션)
- PT 에디토리얼 2단 배너, Best 랭킹 그리드, 타입별 쇼케이스 그리드.

## 토큰 매핑 (29cm 관찰값 → 우리 토큰)
- 폭: 콘텐츠 full-bleed 히어로 / 본문 max-w-[1280px].
- 타이포: 굵은 산세리프 대형 제목 + 뮤트 서브카피 → 기존 Pretendard/ink·mute 그대로.
- 색: 모노톤 + 할인 레드 → 기존 ink/paper/mute/line/sale 그대로. 신규 색 없음.
- 상품 행 할인 표기: `28%` 레드 + 가격 볼드 → 기존 `sale`/`won` 유지.
